# Vehicle-Service-Management

A Python and MySQL based Vehicle Service Management System that manages customers, vehicle servicing, spare parts inventory, smart service recommendations, and PDF bill generation.

## 📌 Overview
**Automates vehicle service center workflow.** Stores **customer & vehicle details**, **tracks service history**, **manages spare parts stock**, and **generates PDF bills**. Provides **smart service recommendations** based on **vehicle usage and age**.

## ✨ Features
- Customer management (add, view, search, update KM)
- Service booking and tracking
- Spare parts inventory with stock alerts
- Smart service recommendations
- PDF bill generation with GST

## 🛠️ Tech Stack
- Python 3.x
- MySQL
- mysql-connector-python
- ReportLab

## 🚀 Quick Start
```bash
git clone https://github.com/ishurawat005-stack/Vehicle-Service-Management.git
pip install mysql-connector-python reportlab
python main.py
```

## 📂 Modules
- **Customer**: add, view, search, update KM
- **Service**: book, update, history
- **Parts**: add, view, update stock
- **Smart Service**: recommendations
- **PDF Bill**: invoice generation
